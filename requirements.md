# Diff Viewer

## 概要

ローカルのGitリポジトリの差分をさっと確認するためのmacOSネイティブアプリ。
複数のリポジトリを監視対象として登録しておき、グローバルショートカット（Command + Control + D）でいつでもdiffを確認できる。

自分専用のツールなので、設定項目や汎用性は不要。全て決め打ちで作る。

### 使い方

1. 設定ファイルに監視したいリポジトリのパスを登録しておく
2. 別のアプリで作業中にショートカットキーを押す
3. diff viewerが前面に表示され、全リポジトリの変更状況を一覧で確認できる
4. もう一度ショートカットキーを押すと閉じる

## 機能

### ランチャーモード

- アプリはランチャーモードでのみ動作する（通常のウィンドウモードは不要）
- グローバルショートカット `Command + Control + D` でトグル表示
- 他のアプリがフルスクリーンでも、ウィンドウを切り替えずにオーバーレイ的に表示される
- ウィンドウサイズは画面の80%、画面中央に固定表示
- 参考: [Nani翻訳のランチャーモード](https://nani.now/ja/blog/launcher-mode)

### diff表示

- 未ステージの変更とステージ済みの変更の両方を表示する
- 表示形式はside-by-side（左右に並べる）
- UIはGitHubのDarkモードに寄せたデザイン
- 変更があるリポジトリだけセクションとして表示する（リポジトリ名がセクションヘッダー）
- 各セクション内ではファイルごとにdiffを表示（折りたたみ可能）
- 全リポジトリに変更がない場合は「変更なし」メッセージを表示

### 監視対象の管理

- 監視対象のリポジトリは設定ファイルにパスのリストとして記述する
- 設定ファイルの場所: `~/.config/diff-viewer/config.json`
- アプリ内のUIでの追加・削除機能は不要

### 更新タイミング

- アプリ起動時（ショートカットで表示されたタイミング）に最新のdiffを読み込む
- 自動更新やリアルタイム監視は不要

## 技術的な方針

- 言語: Swift
- フレームワーク: SwiftUI（macOS向け）
- macOS 26.3.1以降で動作すればよい（後方互換は不要）
- 外部公開しない自分専用ツール
- Gitのdiff取得は `git diff` / `git diff --staged` コマンドの実行結果をパースする
- ビルド・開発はXcodeを使用
